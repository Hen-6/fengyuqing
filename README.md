## 名字的由来

**风雨情**，取自苏轼《定风波·莫听穿林打叶声》：

> 莫听穿林打叶声，何妨吟啸且徐行。竹杖芒鞋轻胜马，谁怕？一蓑烟雨任平生。
> 料峭春风吹酒醒，微冷，山头斜照却相迎。**回首向来萧瑟处，归去，也无风雨也无晴。**

"也无风雨也无晴"——风雨与晴天皆不在意，心境超然，不为外物所动。此境此情，便是风雨情。

> 注："情"在古汉语中与"诚"相通，有真实、真诚之意（通假字）。诗词中"情"字无处不在：故园情、离别情、山水情、思乡情……风雨无情，诗词有心。

## 为什么是"风雨"

"风雨"是中国古典诗词最核心的意象之一。天地无常，人生聚散，风雨既是自然之象，也是命运之隐。

| 时代 | 诗句 | 出处 |
|------|------|------|
| 先秦 | 风雨凄凄，鸡鸣喈喈。既见君子，云胡不夷。 | 《诗经·郑风·风雨》 |
| 唐 | 夜来风雨声，花落知多少 | 孟浩然《春晓》 |
| 唐 | 随风潜入夜，润物细无声 | 杜甫《春夜喜雨》 |
| 唐 | 溪云初起日沉阁，山雨欲来风满楼 | 许浑《咸阳城东楼》 |
| 宋 | 昨夜雨疏风骤，浓睡不消残酒 | 李清照《如梦令》 |
| 宋 | 已是黄昏独自愁，更著风和雨 | 陆游《卜算子·咏梅》 |
| 宋 | 莫听穿林打叶声，何妨吟啸且徐行 | 苏轼《定风波》 |
| 宋 | 可惜流年，忧愁风雨，树犹如此 | 辛弃疾《水龙吟》 |

## 功能

### 🌸 飞花令
选择一个关键字，系统展示一句含该字的诗，你来接出另一句。帮助积累诗词储量，不限对答次数。

### 🔗 接龙
系统出上句，你接下句，要求：末字相同、诗句在库、≥4字。锻炼对仗与记忆。

### 🌺 寻花令
100字提示格，猜出五言或七言对句。绿色=位置正确，黄色=存在但位置错误，灰色=不存在。

## 熟悉度体系

| 等级 | 名称 | 说明 |
|------|------|------|
| 1 | 陌生 | 完全不认识 |
| 2 | 认字 | 见过诗题/作者 |
| 3 | 识句 | 能找出一句以上 |
| 4 | 成篇 | 能背诵整首 |
| 5 | 全知 | 知道作者/背景/典故 |

游戏答对自动升至 Level 3，基于 Duolingo HLR 算法安排复习遗忘曲线。
